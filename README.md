# 🔐 Password Generator - Day 1 Project 31

## 📌 Project Overview

This project is a modern **Password Generator Application** created as part of my semester challenge to build 200 websites.

It allows users to generate secure random passwords and copy them instantly using JavaScript.

* ⚡ Instant Password Display
* 🎨 Clean and Simple UI
